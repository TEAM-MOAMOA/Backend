# 모아모아 
서울 우먼 테크 해커톤 프로젝트 Backend API 

## Description
- Node v20.5.0
- [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

* [local] Swagger: http://localhost:${PORT}/api
* [local] API root: http://localhost:${PORT}/

## Installation
```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
